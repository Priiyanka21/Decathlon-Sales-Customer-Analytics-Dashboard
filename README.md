# Decathlon Sales & Customer Analytics Dashboard

**Smarter insights. Stronger connections. Better performance.**

An interactive analytics dashboard built to explore sales performance, customer behavior, and product trends for a sporting goods retailer.

## 📊 Overview

This dashboard provides a 360° view of business performance, combining high-level KPIs with detailed breakdowns across sports categories, product lines, customer retention, and demographics.

## 🔑 Key Metrics

- **Total Sales:** ₹421M
- **Total Orders:** 30K
- **Total Customers:** 9K
- **Average Order Value:** ₹14.02K

## 🏆 Top 5 Sports Categories by Sales

| Rank | Category   | Sales   |
|------|------------|---------|
| 1    | Cycling    | ₹92.2M  |
| 2    | Outdoor    | ₹47.4M  |
| 3    | Running    | ₹40.3M  |
| 4    | Gym        | ₹38.3M  |
| 5    | Badminton  | ₹36.1M  |

## 📈 Dashboard Sections

- **Sales Over Time** — Daily order value trends across the year
- **Sales by Product Category** — Comparison across Cycling, Camping, Running, Fitness, Trekking, and Cricket
- **Customer Retention Rate** — Monthly retention trends
- **Sales by Gender** — Nearly even split between male (49%) and female (50%) customers

## 🛠️ Tools Used

- Power BI (data modeling, DAX measures, visuals)
- Data cleaning and transformation via Power Query

## 💡 Insights

- Cycling is the clear leader in category sales, nearly double the next closest category
- Customer base is evenly split by gender, suggesting broad market appeal
- Retention rate fluctuates seasonally, with peaks around March and July
